# PPA因子筛选标准

## 核心指标

| 指标 | 要求 | 说明 |
|------|------|------|
| **PPC** | < 0.5 | Power Pool Correlation，必须低于0.5否则直接拒绝 |
| **Sharpe** | > 1.0 | 样本内建议 ≥1.05 更安全 |
| **Fitness** | > 0.5 | 稳定性指标，低于0.5不考虑 |
| **Margin** | > Turnover | 赚钱效率必须覆盖交易成本 |

## 额外要求

1. **实盘稳健性**：实盘Sharpe不能显著低于样本内，实盘 ≥ 样本内最优，最多小幅下降
2. **中性化**：必须做行业中性/市场中性，不中性极易导致高相关性、高回撤
3. **年度均匀**：年度收益尽量均匀，不要极端依赖某几年

## 新手避坑

- Sharpe卡1.00、Fitness卡0.5风险极高，留安全边际
- 只看样本内，不看实盘与年度稳定性
- 频繁小改参数重复提交，容易被判定相似因子
- 忽视相关性，PPC/PC过高依然会被拒
- 追求高收益忽略回撤，PPA更看重稳健、可持续、低拥挤

## 合格标准

1. PPC < 0.5
2. Sharpe > 1.0
3. Fitness > 0.5
4. Margin > Turnover
5. 实盘稳健不下滑
6. 行业中性开启
7. 年度表现均匀
8. 低相关性、低过拟合

## 本质理解

PPA不是越复杂越好，简单逻辑 + 稳定信号 > 复杂堆砌。宁可收益低一点，也不要高相关性。
