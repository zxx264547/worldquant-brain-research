## 浏览器翻页修复

Alpha页面不能翻页时：
1. F12打开开发者工具
2. 切换到控制台(Console)
3. 输入：
```javascript
document.querySelector('.footer__button.footer__button--next').click()
```

## TARE Token限制解决

使用MCP + OCR实现无人值守：
1. 安装Tesseract OCR
2. 截图输入框保存为input_box.png
3. 运行监控脚本自动填写
