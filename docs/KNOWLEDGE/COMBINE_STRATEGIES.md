# Combine组合策略

## 核心思路

- **看OS下菜碟**：根据OS排名调整提交策略
- **不要死磕同一个Theme**：灵活切换
- **组合不同类型的Alpha分散风险**

## 高效策略

| OS状态 | 策略 |
|--------|------|
| OS < 0.5 | 去交其他区域Alpha稳住收益 |
| OS > 0.5 | 交Theme加成收益翻倍 |

## 进阶技巧

1. **低相关性 + 高fit = 硬通货**
2. 不要只追求高分，要追求不同方向
3. 年度表现均匀比单年暴涨更重要

## Combine为负数怎么办

1. 分析负分原因：相关性太高？fit太低？
2. 检查提交因子的质量
3. 参考其他顾问的经验调整策略
