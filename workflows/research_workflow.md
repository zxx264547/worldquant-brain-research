# 研究主流程

## 完整研究流程

```
┌─────────────────────────────────────────────────────────────┐
│                    Alpha研究工作流                            │
└─────────────────────────────────────────────────────────────┘

1. 数据准备
   │
   ├─ 下载数据集到 data/datasets/
   ├─ 运行 webdata_raw_parser.py 解压
   └─ 分析字段命名规律

2. Alpha挖掘
   │
   ├─ batch_mining.py 启动挖掘
   ├─ 设置筛选参数
   └─ 并发控制(10并发)

3. 多轮筛选
   │
   ├─ screening_pipeline.py
   ├─ PPA标准: PPC<0.5, Sharpe>1.0, Fitness>0.5
   └─ Margin > Turnover

4. 相关性去重
   │
   ├─ correlation_analysis.py
   ├─ 分族 + 组内排序
   └─ 每族保留1-2条代表

5. 续航力评分
   │
   ├─ os_score_calculator.py
   ├─ D1-D4四维度
   └─ 75+ STRONG BUY

6. 提交前检查
   │
   ├─ 行业中性/市场中性
   ├─ 年度收益均匀性
   └─ 确认无相似因子

7. 提交入库
   │
   ├─ 记录到 factor_library.db
   ├─ 保存表达式到 factors/
   └─ 更新每日日志
```

## 每日工作流程

| 时间 | 任务 |
|------|------|
| 早间 | 查看OS排名和前日表现 |
| 日间 | 执行Alpha挖掘任务 |
| 晚间 | 整理当日结果，更新日志 |

## 关键检查点

### 提交前检查清单
- [ ] PPC < 0.5
- [ ] Sharpe >= 1.0
- [ ] Fitness > 0.5
- [ ] Margin > Turnover
- [ ] 已做中性化
- [ ] 年度收益均匀
- [ ] 无高度相似因子

### OS策略
- [ ] OS < 0.5: 交其他区域Alpha
- [ ] OS >= 0.5: 交Theme
