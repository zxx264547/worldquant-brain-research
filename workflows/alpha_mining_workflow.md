# Alpha挖掘标准化流程

## 流程图

```
数据字段
    │
    ▼
┌─────────────────┐
│  生成变体        │  生成20+变体表达式
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  首轮筛选        │  Sharpe >= 0.7
│  min_sharpe=0.7 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  次轮筛选        │  Sharpe >= 0.7
│  min_sharpe=0.7 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  第三轮筛选       │  Sharpe >= 1.0
│  min_sharpe=1.0 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Fitness筛选     │  Fitness >= 0.5
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  相关性分族       │  阈值0.8
│  去重            │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  续航力评分      │  75+ STRONG BUY
└────────┬────────┘
         │
         ▼
     可提交Alpha
```

## 参数配置

```python
CONFIG = {
    "min_sharpe_step1": 0.7,    # 首轮
    "min_sharpe_step2": 0.7,    # 次轮
    "min_sharpe_step3": 1.0,    # 第三轮
    "min_fitness": 0.5,
    "max_turnover": 0.7,
    "correlation_threshold": 0.8,
    "batch_size": 10,
    "max_variants": 20,
    "concurrent_workers": 10,
}
```

## 变体生成策略

1. **参数变化**: 调整时间窗口参数
2. **算子组合**: 尝试不同算子组合
3. **数据字段**: 同一字段的不同表达
4. **中性化**: 添加/移除中性化
