# 提交前检查清单

## Alpha提交标准

### 核心指标检查

| 检查项 | 标准 | 权重 |
|--------|------|------|
| PPC | < 0.5 | 核心 |
| Sharpe | >= 1.0 | 核心 |
| Fitness | > 0.5 | 核心 |
| Margin | > Turnover | 核心 |

### 质量检查

| 检查项 | 说明 |
|--------|------|
| [ ] 中性化 | 必须做行业中性/市场中性 |
| [ ] 年度均匀 | 年度收益不要极端依赖某几年 |
| [ ] 相关性 | 避免与已有因子高度相关 |
| [ ] 稳定性 | 实盘Sharpe不能显著低于样本内 |

### 提交策略

#### OS < 0.5 时
- 提交其他区域的Alpha
- 避免交Theme（Buff浪费）

#### OS >= 0.5 时
- 可以交Theme
- 高OS + Theme加成收益翻倍
